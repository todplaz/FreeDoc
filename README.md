Apprendre Rails et la gestion des BDD via une app de prise de RDV médicaux
* Gestion des models
* Gestion des migrations

Pour tester la BDD :
* Cloner le repo
* `bundle install`
* `rails db:migrate`
* `rails db:seed`
* `rails c`
* Faire les tests nécessaires (`tp Doctor.all`, `tp Patient.all`, `tp JoinTableDoctorSpecialty.all`...)
